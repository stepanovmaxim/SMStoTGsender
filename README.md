A simple and reliable Android app to automatically forward incoming SMS messages to a Telegram bot. Perfect for receiving bank alerts and 2FA codes on another device. </br> Supports sender filtering. All settings (Token & ID) are stored locally on your device for privacy.

Setup Guide:</br>
Create a Bot: Open Telegram, find @BotFather, send /newbot and copy the API Token.</br>
Create new group and add your new bot.</br>
Get your ID: Find @getidsbot, add it to your group, press Start, and copy your Id, remove this bot after that.</br>

In the App:</br>
Paste the Token and Chat ID into the fields.</br>
(Optional) In the "Filter" field, enter sender names separated by commas (e.g., Bank, PayPal, Amazon). Leave empty to forward all SMS.</br>
Click Save Settings.</br>
Test: Click Send Test Message. If you receive it in Telegram, everything is ready!</br>

Простое и надежное приложение для автоматической пересылки входящих SMS в Telegram. Позволяет читать банковские уведомления и коды подтверждения (2FA) с другого устройства. </br>
Поддерживает фильтрацию по отправителю. Ваши данные (Токен и ID) хранятся только на телефоне.

Инструкция по настройке:</br>
Создайте бота: Откройте Telegram, найдите @BotFather, отправьте команду /newbot и получите API Token.</br>
Создайте группу в Telegram и добавьте туда своего нового бота.</br>
Узнайте свой ID группы: Найдите бота @getidsbot, добавьте его в группу, нажмите Start и скопируйте Id, удалите этого бота из группы.</br>

В приложении:</br>
Вставьте Токен и Chat ID в соответствующие поля.</br>
(Опционально) В поле "Фильтр" напишите имена отправителей через запятую (например: Sberbank, 900, Google). Оставьте пустым, чтобы получать все SMS.</br>
Нажмите Сохранить настройки.</br>
Проверка: Нажмите Отправить тестовое сообщение. Если оно пришло в Telegram — всё работает!</br>
